# Food AI Static Site Template

這個資料夾是最小可部署的靜態網站模板。

目前包含：
- `index.html`
- `privacy.html`
- `terms.html`
- `support.html`
- `en/index.html`
- `en/privacy.html`
- `en/terms.html`
- `en/support.html`
- `styles.css`

使用方式：
1. 先替換以下 placeholder：
- `foodie.eye.2025@gmail.com`
- `https://sean4437.github.io/food-ai-site`
- App Store 正式連結若尚未取得，先用 `#`
2. 把整個資料夾內容複製到網站 repo root。
3. 依 `../pages-deploy-guide.md` 上 GitHub Pages 或 Cloudflare Pages。

備註：
- 這份模板先追求可上線，不含後端。
- 根目錄為繁體中文，`/en/` 為英文版骨架。
- 若要做正式品牌站，再基於這組 HTML/CSS 延伸即可。
